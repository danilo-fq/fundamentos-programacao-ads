# Exercícios de Programação de Computadores

Este repositório contém os exercícios da disciplina **Programação de Computadores**, ministrada pela professora **Joanacelle Caldas De Melo**. Destinado à turma **P1B** do curso de **Análise e Desenvolvimento de Sistemas** da faculdade **Unipê**, turno da noite.

## Estrutura do Repositório

Os exercícios estão organizados em pastas separadas por aulas, facilitando a navegação e o acompanhamento do progresso ao longo do curso.

### Pastas de Aulas

- **Aula_04_python_basico/**: Exercícios básicos de Python.

## Como Usar

1. Navegue até a pasta da aula desejada.
2. Abra os arquivos de exercício correspondentes.
3. Execute os códigos conforme as instruções de cada exercício.

## Como Testar os Códigos

Para testar os códigos, é necessário ter o Python instalado na máquina. No terminal, navegue até a pasta do arquivo e execute um dos comandos abaixo, dependendo do seu sistema operacional:

- **Windows**:
  ```
  py nome-do-arquivo.py
  ```

- **Linux**:
  ```
  python3 nome-do-arquivo.py
  ```

Substitua `nome-do-arquivo.py` pelo nome do arquivo que deseja executar.
